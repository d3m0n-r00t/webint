# webint
Domain OSINT tool written in python. 

```
git clone https://github.com/d3m0n-r00t/webint.git
```

## Working
```
python3 webint.py <domianname>
```
Example
```
python3 webint.py example.com
```

### Features
1. List Subdomains
2. WHOIS data
3. Reverse WHOIS 
4. Reverse IP lookup
5. IP history
6. Currnent IP Location
